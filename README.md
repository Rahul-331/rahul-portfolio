# 🚀 My Portfolio

A modern, responsive portfolio website built with React and Vite, showcasing projects, skills, and experience with smooth animations and beautiful design.

## 🌐 Live Demo

**[View Portfolio Live](https://rahul-portfolio-hipu282xm-rahul-331s-projects.vercel.app/)**

## ✨ Features

- **Responsive Design** - Mobile-first approach that works seamlessly on all devices
- **Smooth Animations** - Engaging transitions and animations powered by Framer Motion
- **Modern Stack** - Built with React 19, Vite, and Tailwind CSS for optimal performance
- **Icon Integration** - Rich set of icons using react-icons
- **Smooth Scrolling** - Navigate sections with react-scroll for a polished experience
- **SEO Friendly** - Clean semantic HTML structure
- **Fast Performance** - Optimized builds and lightning-fast development experience

## 📋 Sections

- **Hero** - Eye-catching introduction with call-to-action
- **About** - Personal background and professional journey
- **Skills** - Technical expertise and competencies
- **Projects** - Showcase of key projects with descriptions
- **Contact** - Easy ways to get in touch

## 🛠️ Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd my-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`

## 🚀 Usage

### Development
```bash
npm run dev
```
Runs the app in development mode with hot module replacement.

### Build
```bash
npm run build
```
Creates an optimized production build in the `dist` folder.

### Preview
```bash
npm run preview
```
Preview the production build locally before deployment.

### Lint
```bash
npm run lint
```
Run ESLint to check code quality.

## 💻 Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: react-icons
- **Scrolling**: react-scroll
- **Linting**: ESLint

## 🎨 Customization

Edit the component files in `src/components/` to customize:
- Update text content and information
- Modify colors in Tailwind CSS classes
- Adjust animation configurations in Framer Motion
- Add or remove sections as needed

## 📦 Deployment

The site can be easily deployed to:
- **Vercel** - `npm i -g vercel` then `vercel`
- **Netlify** - Connect your GitHub repo for automatic deployments
- **GitHub Pages** - Build and push to `gh-pages` branch
